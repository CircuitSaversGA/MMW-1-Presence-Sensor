Advanced, 24GHz mmWave human presence detection for Home Assistant. 

## 🚀 The Mission
Standard PIR motion sensors leave you in the dark. The **MMW-1** uses micro-radar to detect presence, not just motion. Whether you're sleeping, reading, or working at a desk, the MMW-1 keeps your smart home aware.

## 💻 Code & Configuration
The MMW-1 is 100% local and open-source.
1. Copy the code from `/esphome/mmw-1-basic.yaml`.
2. Flash your ESP32-C3 using the ESPHome dashboard.
3. Native discovery in Home Assistant will handle the rest.
